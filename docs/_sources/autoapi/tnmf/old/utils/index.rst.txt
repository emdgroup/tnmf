   :returns:

.. function:: shift(arr: numpy.ndarray, step: int) -> numpy.ndarray

   Shifts the given array a certain number of steps along the last axis, filling the empty places with zeros.

   :param arr: The array to be shifted.
   :type arr: np.ndarray
   :param step: The number of steps the array shall be shifted.
   :type step: int

   :returns:
   :rtype: The shifted (and zero-padded) array.
